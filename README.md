# GE2RBConvertTool
> 这是一款用于《噬神者2：狂怒解放》（God Eater 2: Rage Burst）存档在 **Steam** 与 **PlayStation** 平台之间自由转换的便捷工具。

---

## 📖 项目简介

本工具旨在帮助玩家在 **Steam 版** 和 **PS Vita / PS TV / PS4 版**（通用 PS 存档格式）之间无缝迁移游戏存档。无需手动修改十六进制，只需拖拽文件夹，即可自动完成解密、平台补丁转换、校验和修复及重新加密的全流程。

本工具由AI编写，经测试无巨大bug。

未完成内容：PS平台角色卡片转换至Steam平台后无法正常加载，Steam平台转换为PS平台不受影响。

---

## ✨ 主要特性

- 🚀 **一键转换**  
  将包含 `SECURE.BIN` 的文件夹直接拖拽到程序图标上，或通过命令行传入路径，所有操作自动完成。

- 🧠 **智能识别平台**  
  根据 `SECURE.BIN` 文件大小自动判断来源平台
  并自动确定转换方向（Steam → PS 或 PS → Steam）。

- 🔒 **安全输出，原档不变**  
  转换后的文件输出到与原文件夹同级的新目录，命名格式为 `[原文件夹名]_converted_[目标平台]`，原始文件始终保持不变。

- 📦 **完全自包含**  
  所有转换规则（包括 `SECURE_convent.txt` 和 `CXX__convent.txt`）均已嵌入可执行文件，**无需任何外部配置文件或依赖库**，开箱即用。

- 📋 **详细日志，窗口保留**  
  转换过程中输出完整操作记录（解密、补丁应用、校验和修复等），完成后窗口保持打开，方便用户确认结果。

- 📁 **全文件覆盖**  
   除主存档 `SECURE.BIN` 外，自动处理所有角色卡片文件（`C*.BIN`），确保整套存档文件完全兼容目标平台。

-   **未完成内容**  
   PS平台角色卡片转换至Steam平台后无法正常加载，Steam平台转换为PS平台不受影响。

## 🚀 使用方法

### 方式一：拖拽操作（推荐）
1. 将包含 `SECURE.BIN` 的文件夹（例如 `PCSH001330fc1c45531120001000`）**拖拽**到 `GE2RBConvertTool.exe` 图标上。
2. 松开鼠标，工具将自动运行，完成后会在同级目录生成转换后的文件夹。

### 方式二：命令行
```cmd
GE2RBConvertTool.exe <文件夹路径>
```
